# NetLogo Resources

A curated list of NetLogo resources: libraries, guides, tools, and more.

---

## Libraries

### [NetLogo-Shapes](https://github.com/danielvartan/logoshapes)

NetLogo-Shapes is a library of NetLogo vector graphics. It provides scalable,
editable versions of NetLogo's iconic shapes, making it easy to customize
visuals for simulations and educational materials without loss of quality.

---

## Guides

### [Agent-Based Evolutionary Game Dynamics](https://wisc.pb.unizin.org/agent-based-evolutionary-game-dynamics/)

A guide to implement and analyze Agent-Based Models within the framework of
Evolutionary Game Theory.

**Authors:** Luis R. Izquierdo; Segismundo S. Izquierdo; William H. Sandholm

## Contributing

To add a new resource, edit [`_data/resources.yml`](_data/resources.yml) and add an entry with the following fields:

```yaml
- title: "Resource Title"
  description: "A brief description of the resource."
  url: "https://example.com"
  category: "Category"          # e.g., Libraries, Guides, Tools
  authors:                      # optional
    - "Author Name"
```
